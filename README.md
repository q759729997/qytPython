
# Python工具包

## 安装方式

- 下载此项目，进入项目根目录，执行如下命令安装：

```python
python setup.py install
```

- 使用pip进行安装

```python
pip install qytPython
```

## 适用平台

- Linux
- Windows

## Python版本要求

- 大于等于`python 3.6`
